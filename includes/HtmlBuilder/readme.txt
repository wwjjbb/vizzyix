HtmlBuilder is a 3rd-party header-only library for generating HTML. This readme
file identifies the origin and version of the library sources as included in
Vizzyix. Filenames ".license" have been added to provide SPDX license details
for the original source files, avoiding the need to edit these.

The license text included in this directory by the original author matches the
wording of the SPDX license, MITNFA, but additionally contains the author's
copyright statement.
